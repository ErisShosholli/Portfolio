# Eris Shosholli Portfolio

Clean portfolio site for a DevOps Engineer and Full-Stack Developer.

## Tech

- Next.js
- TypeScript
- CSS
- lucide-react

## Local development

```bash
pnpm install
pnpm dev
```

Open `http://127.0.0.1:3000`.

## Production build

```bash
pnpm build
```

## Deploy to Vercel

Import the repository in Vercel. Vercel will detect Next.js automatically.

- Install command: `pnpm install`
- Build command: `pnpm build`
- Output: handled by Next.js
